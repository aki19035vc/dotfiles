# dotfiles

## dotifileを配置方法

```
deploy.sh
```

※ Brewfileも配置されます

## homebrewのインストール
1. xcodeをインストール

2. homebrewをインストール
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```

3. Brewfileからアプリをインストール
```
brew bundle
```

## iterm2のプロファイル設定方法

`Preferences > Profiles > Colors > Color Presets`でアクセスし、`Snazzy.itermcolors`をインポートする
